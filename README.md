# 품질관리 대시보드

건설현장 품질관리 시스템 웹 대시보드

## 기능
- 실시간 현장 현황 모니터링
- 품질관리자 배치기준 관리
- 품질검사 결과 차트
- 품질관리 도우미 챗봇

## 파일 구조
```
├── index.html          # 메인 HTML
├── css/
│   └── styles.css      # 스타일시트
├── js/
│   ├── data.js         # 데이터
│   ├── charts.js       # 차트 관련
│   ├── modals.js       # 모달 관련
│   ├── utils.js        # 유틸리티
│   └── main.js         # 메인 로직
└── icons/              # 아이콘 이미지
```
